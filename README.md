# Study-Tabs
📚 Study Tabs - Chrome Extension

Turn your browsing into a productivity game 🎯

Chrome Extension | JavaScript

✨ Features

✅ Earn XP for study-related tabs (coding, notes, docs)
✅ Lose XP for distractions (YouTube, reels, random browsing 😭)
✅ Real-time XP tracking system
✅ Level-based motivation system
✅ Automatic tab analysis (no manual input)
✅ Clean and minimal UI

🎮 Demo

"leetcode problem" → 🟢 +10 XP
"physics notes" → 🟢 +10 XP
"youtube gaming" → 🔴 -5 XP
"instagram reels" → 💀 -10 XP

👉 Your tabs decide your progress

🚀 Install (30 seconds)

chrome://extensions/

Turn ON Developer Mode

Click Load Unpacked

Select project folder

Pin extension → Start using

📊 Example Gameplay
Tab Title	Action	XP Change
leetcode problem	🟢 Productive	+10 XP
physics notes	🟢 Study	+10 XP
youtube gaming	🔴 Distracting	-5 XP
instagram reels	💀 Time Waste	-10 XP
🧠 Concept

Study Tabs converts your daily browsing into a simple reward system.
Instead of losing time unknowingly, you get instant feedback on your habits.

👉 Productive tabs → Gain XP
👉 Distractions → Lose XP

Small changes → Better focus over time
